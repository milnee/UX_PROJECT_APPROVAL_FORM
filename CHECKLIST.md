# COM321 UX Project - Requirements Checklist

## Deliverable 1: Web Form Submission [30 marks]

### Technical Requirements
- [x] **HTML5** - Form uses HTML5 semantic elements and form inputs
- [x] **CSS3** - Custom CSS file (styles.css) with modern styling
- [x] **JavaScript** - Interactive functionality (script.js) for form validation and calculations

- [x] **Travel Details Section** - Captures destination, purpose, departure/return dates, overnight stay checkbox

- [x] **Declaration Section** - Confirmation checkboxes for:
  - [x] No travel arrangements made before approval
  - [x] Information accuracy confirmation

- [x] **Cost Calculation** - Automatic total calculation as user enters costs

- [x] **Save Draft** - LocalStorage functionality to save form progress

### Code Quality
